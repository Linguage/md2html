## 经典数学公式

### 欧拉恒等式
被誉为"最美的数学公式"：

**e^(iπ) + 1 = 0**

这个公式连接了数学中最重要的五个常数。

### 微积分基本定理

∫[a→b] f'(x)dx = f(b) - f(a)

## 数列与级数

### 斐波那契数列
F(n) = F(n-1) + F(n-2)

```javascript
function fibonacci(n) {
  if (n <= 1) return n;
  return fibonacci(n - 1) + fibonacci(n - 2);
}

// 生成前10项
for (let i = 0; i < 10; i++) {
  console.log(`F(${i}) = ${fibonacci(i)}`);
}
```
