## 代码示例

```python
import math

def calculate_distance_to_star(parallax):
    """根据视差计算恒星距离"""
    return 1 / parallax  # 单位：秒差距

# 计算比邻星的距离
proxima_distance = calculate_distance_to_star(0.768)
print(f"比邻星距离: {proxima_distance:.2f} 秒差距")
```
