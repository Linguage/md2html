# 粒子特效演示

这是一个带有粒子背景特效的 Markdown 示例。

- 支持多种主题特效
- 可在 Markdown 顶部用注释标记特效类型
- 转换后 HTML 自动带有特效 class 和动画

```js
console.log('Hello, Theme Effects!')
```
